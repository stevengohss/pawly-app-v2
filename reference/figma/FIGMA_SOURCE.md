## Source-of-truth rules

1. Live Figma is the exact visual source of truth.
2. /docs governs product behaviour, scope, architecture and data.
3. Figma component pages must be implemented as reusable production components.
4. Existing Pawly V1 screen code must not be used as a visual reference.
5. Do not replace Figma icons with generic alternatives where exact assets are available.
6. Do not redesign approved screens.
7. Validate every screen visually before proceeding to the next.
